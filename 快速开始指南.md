# 币安期货自动交易系统 - 快速开始指南

## 🚀 快速开始（5分钟上手）

### 1. 启动程序
```bash
python3 eth_fakeout_gui.py
```

### 2. 登录系统
1. 在 **🔐 登录** 标签页输入币安API Key和Secret
2. 勾选"保存凭证"（可选）
3. 点击"登录系统"

### 3. 启动策略（推荐流程）
1. 切换到 **🎯 标的选择** 标签页
2. 选择"自动（综合评分）"模式
3. 点击"🔄 刷新合约列表"
4. 点击"✅ 应用选择"

5. 切换到 **🎮 手动控制** 标签页
6. 确认"模拟模式"已勾选（重要！）
7. 点击"▶️ 启动策略"

### 4. 观察运行
1. 切换到 **📊 市场监控** 标签页
2. 观察实时日志
3. 查看系统状态和市场指标
4. 查看发现的信号（**💡 假突破信号** 标签页）

### 5. 调整参数（可选）
1. 切换到 **⚙️ 参数配置** 标签页
2. 根据需要调整参数
3. 点击"💾 保存并应用"
4. 参数会立即应用到运行中的系统

### 6. 手动干预（可选）
1. 在 **🎮 手动控制** 标签页
2. 可以暂停/恢复/停止策略
3. 可以手动开仓/平仓
4. 可以查看和管理持仓

## 🎯 常见场景

### 场景1：保守交易（降低风险）
在 **⚙️ 参数配置** 标签页调整：
- 最大回撤：5%（降低）
- 最大连续亏损：2次（减少）
- 每日亏损限制：30 USDT（降低）
- 最小盈亏比：3.0（提高）
- 突破确认K线数：3（增加）

### 场景2：激进交易（提高收益）
在 **⚙️ 参数配置** 标签页调整：
- 最大回撤：15%（提高）
- 最大连续亏损：5次（增加）
- 每日亏损限制：100 USDT（提高）
- 最小盈亏比：1.5（降低）
- 突破确认K线数：1（减少）

### 场景3：手动开仓
1. 在 **🎮 手动控制** 标签页
2. 输入合约（如ETHUSDT）
3. 选择方向（做多/做空）
4. 输入数量（如100 USDT）
5. 设置止损（如2%）和止盈（如4%）
6. 点击"🟢 手动做多"或"🔴 手动做空"
7. 确认交易

### 场景4：紧急平仓
1. 在 **🎮 手动控制** 标签页
2. 点击"🔄 刷新持仓"
3. 查看持仓列表
4. 点击"❌ 全部平仓"或"❌ 平选中的"
5. 确认平仓

## ⚠️ 重要提醒

### 首次使用
1. ✅ **必须先在模拟模式下充分测试**
2. ✅ 理解每个参数的含义
3. ✅ 观察策略在模拟模式下的表现
4. ✅ 只有充分测试后才切换到实盘模式

### 切换到实盘模式
1. 取消勾选"模拟模式"
2. 确认所有参数设置合理
3. 确认账户资金充足
4. **从小仓位开始测试**

### 参数调整建议
1. 不要频繁调整参数
2. 每次只调整1-2个参数
3. 调整后观察一段时间（至少几小时）
4. 根据实际表现优化

## 📊 监控重点

### 每日检查
1. **总盈亏**: 是否在预期范围内
2. **最大回撤**: 是否超过设定值
3. **连续亏损**: 是否接近触发熔断
4. **交易次数**: 是否符合预期

### 实时监控
1. **系统状态**: 确保正常运行
2. **市场状态**: 了解当前市场环境
3. **信号质量**: 观察信号的置信度
4. **持仓情况**: 关注未实现盈亏

## 🔧 故障排查

### 问题：策略无法启动
**解决方法**：
1. 检查是否已登录
2. 检查网络连接
3. 查看实时日志中的错误信息

### 问题：没有发现信号
**正常情况**：
1. 策略只在高置信度时才会发出信号
2. 市场可能处于不活跃状态
3. 可以调低"最小信号置信度"参数（不推荐）

### 问题：交易未执行
**检查项**：
1. 是否是模拟模式？
2. 是否触发了风险管理限制？
3. 查看实时日志中的具体原因
